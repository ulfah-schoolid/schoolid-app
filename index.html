Bantuan

Bagaimana cara menambah murid?
+
Buka Profil → Pengaturan Akun → Kelola Murid → tap tombol "+ Tambah" di pojok kanan atas.
Bagaimana cara input absensi?
+
Dari Dashboard Guru, tap tombol "Absensi" di Menu Cepat. Pilih status H/S/I/A untuk setiap murid lalu tap Simpan.
Bagaimana cara memberi nilai?
+
Dari Dashboard Guru, tap "Penilaian Cepat". Pilih aktivitas dan aspek, lalu input nilai angka 0-100. Nilai otomatis dikonversi ke BSB/BSH/MB/BB.
Data murid tersimpan di mana?
+
Saat ini data tersimpan di browser (localStorage). Pastikan tidak menghapus cache browser agar data tidak hilang.
Bagaimana cara menghubungi support?
+
Kirim email ke support@schoolid.co.id atau WhatsApp ke 0812-0000-0000.
